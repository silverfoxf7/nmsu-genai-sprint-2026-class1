# my-first-agent-system

This repository is for learning how to build **AI-assisted systems** using [Cursor](https://cursor.com)—an editor where you can pair with AI to write, understand, and run code.

## Project description

This is a small hands-on project for experimenting with agents: asking questions, getting help with code, and practicing a workflow where you and AI tools work together. The focus is on learning, not shipping a finished product.

## Purpose

- Get comfortable using Cursor as part of a development workflow.
- Practice describing goals clearly so AI assistance stays useful and on track.
- Build intuition for what AI can do well (and where you still need to think carefully).

## What I'm Learning

- How to use AI chat and inline help while coding.
- How to break tasks into steps an assistant can follow.
- Basics of version control and keeping a project organized as it grows.

## Next Steps
